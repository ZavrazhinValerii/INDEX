Чем отличаются Web server и Application server?

Web server: -включает только веб-контейнер; -полезен или приспособлен для статического контента; потребляет или использует меньше ресурсов; -организуют среду выполнения для веб-приложений; -многопоточность не поддерживается; -емкость веб-сервера ниже, чем у сервера приложений; -используются протоколы HTML и HTTP. Application server: -включает в себя веб-контейнер, а также контейнер EJB; -приспособлен для динамического контента; -использует больше ресурсов; -организуют среду выполнения для корпоративных приложений; - поддерживается многопоточность; -мощность сервера приложений выше, чем у веб-сервера; -используются протоколы GUI, HTTP и RPC / RMI.
